- [ ] the original signal is preserved

- [ ] at least one artifact is publishable, buildable, reusable, or sellable

## 6. Failure modes

| Failure | Symptom | Fix |
|---|---|---|
| Vibe-only output | Sounds good but cannot be reused | Add schema, tokens, rules, or artifact structure |
| Style drift | Later outputs stop matching source | Add negative constraints and preservation rules |
| No handoff | Developer/designer has to guess | Add component/spec/behavior details |
| No product bridge | Beautiful but unsellable | Add audience, use cases, license, pricing, copy |
