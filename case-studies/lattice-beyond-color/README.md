# Lattice Beyond Color

A Seed-Loom reference package for transferring biomimetic and crystal-lattice principles into product systems, web interfaces, and deterministic code.

## Core rule

Biomimicry is admitted only when the source mechanism changes routing, filtering, resilience, inference, energy, or controlled failure. A biological reference that changes only palette or texture is decorative and does not pass the functional-fidelity gate.

```text
evidence -> mechanism -> formula -> semantic parameter -> behavior -> visible state -> verification
```

## Concepts

- **HelioVeil** — tunable photonic climate skin for solar gain, glare, daylight, and thermal control.
- **EchoVault** — graded phononic privacy and clarity fields.
- **Fault Garden** — evidence graph where vacancies, dislocations, dopants, and grain boundaries expose epistemic conditions.
- **Capillary Commons** — reticulate infrastructure routing with explicit redundancy and failure testing.
- **Nacre Sentinel** — staggered lamellar cyber-resilience architecture for deflection, quarantine, bridging, and verified recovery.

## Included

- `lattice-system.tokens.json` — semantic tokens, parameter ranges, formulas, UI translations, scores, selling points, and production image prompts.
- `lattice-engine.ts` — dependency-free deterministic state derivation and reusable mechanism formulas.
- `lattice-web-demo.html` — standalone interactive visualization for all five concepts.
- `DECISION_LOG.md` — choice hierarchy, explainability rules, and deterministic/probabilistic boundary.

## Run the demo

Open `lattice-web-demo.html` directly in a modern browser. No build step or external dependency is required.

## Strategic position

The moat is not a biomimetic visual style. It is the traceable compiler that translates evidence and equations into semantic parameters, behavior, visible state, code, tests, and replayable decisions.

**Recommended sequence:** use Fault Garden as the flagship software proof and Nacre Sentinel as the commercially direct enterprise proof.
